# requirements.md - 要件定義

## Goal（何を作るか）

ホッキョクグマの環境保護につながるWEBサイトをNext.jsで作成する。

## Must（必須機能）

- [ ] Next.js 14を使用したモダンなWEBサイト
- [ ] ホッキョクグマの現状・課題を伝えるコンテンツ
- [ ] 環境保護への行動を促すUI
- [ ] レスポンシブデザイン（モバイル対応）
- [ ] 基本的なナビゲーション

## Nice（あったら良い機能）

- [ ] 寄付・支援への導線
- [ ] ニュース・更新情報セクション
- [ ] 統計データの可視化
- [ ] 多言語対応
- [ ] アクセシビリティ対応

## Out of scope（対象外）

- ユーザー認証・ログイン機能
- データベース連携（初期段階）
- 決済機能（MVP段階）
- 管理画面

## ユーザー・役割

- **一般ユーザー**: ホッキョクグマの保護に関心のある人
- **目的**: 現状を知り、保護活動に参加したい

## Done条件（受け入れ条件）

1. ローカル環境で`npm run dev`で起動できる
2. ホッキョクグマ保護に関する主要コンテンツが表示される
3. モバイル・デスクトップで適切に表示される
4. 基本的なナビゲーションが機能する
